Интернет-магазин. Регистрация и логин администратора системы. Загрузка товаров из базы, добавление нового продукта. Возможность добавления товара в корзину на странице, отображения отзывов о выбранном товаре и его удаления. Реализована логика отображения элементов, которые доступны только авторизованному пользователю. Работа с данными через API.
